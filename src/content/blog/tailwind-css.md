Dulu anti sama Tailwind, sekarang jadi favorite. Utility-first itu bikin styling cepet dan konsisten.

### Kenapa Tailwind?
- **No context switching** — gak perlu ganti file CSS
- **Consistent spacing** — pake scale yang udah ditentuin
- **Responsive built-in** — `sm:`, `md:`, `lg:` langsung di class
- **Dark mode** — tinggal tambahin `dark:`

```jsx
<div className="p-4 bg-white dark:bg-neutral-900 rounded-xl shadow-md hover:shadow-lg transition-shadow">
```

Dulu nulis CSS, sekarang tinggal compose class. Game changer banget!
